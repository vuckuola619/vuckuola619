### 🦕 Currently Building: **[PaleoBar](https://github.com/vuckuola619/paleobar)**
*Tiny Ecosystem Hero — A cozy, educational idle RPG companion that lives directly in your taskbar.*
* **Dual-Window System**: Tiny compact always-on-top window (340x96px) + Expanded Colony Command Center (900x600px).
* **Zero Audio Files**: Features on-the-fly generative ambient music synthesized using the **Web Audio API** 🎶.
* **Pure Performance**: Built using Tauri 2 (Rust) backend + vanilla HTML/CSS/JS (no framework slop!).
* **Dynamic Simulation**: Includes weather events (like Volcanic Ash!), procedural battles, and educational quizzes.
